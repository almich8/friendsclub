# Клуб Друзей Школы-студии МХАТ — Web Bingo Game

Статическая адаптивная одностраничная игра на vanilla HTML/CSS/JS.

## Запуск локально

1. Откройте папку проекта.
2. Запустите любой статический сервер, например:
   ```bash
   python3 -m http.server 8080
   ```
3. Откройте в браузере `http://localhost:8080`.

## Структура

- `index.html` — разметка интерфейса и модалки;
- `styles.css` — стили, адаптив, анимации;
- `script.js` — логика бинго, localStorage, модалка, линии;
- `logo.svg` — текущий SVG-логотип в header.

## Быстрая замена текстов

- Тексты header/hero/кнопок: `index.html`.
- Бегущая строка: массив `PHRASES` в `script.js`.
- Тексты победного баннера: `updateStats()` в `script.js`.

## Массив описаний для бинго

- Массив `TASK_POOL` в `script.js`.

## Подключение логотипа

- Логотип подключен в `index.html` как `<img src="logo.svg" ...>` внутри `.logo-pill`.
- При наличии оригинального файла замените содержимое `logo.svg` на предоставленный SVG без изменения имени файла.
